# 📖 Budget App <a name="about-project"></a>


This app is for mobile, that allow user to see the amount of cash he spend on different products, the user can add and remove categories and add new transactions associated with a category.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

```sh
 install ruby
 install ruby on rails
 install node.js
```


### Setup

Clone this repository to your desired folder:


```sh
  cd my-folder
  git clone https://github.com/Ellyboi/BudgetApp.git
```


### Install

Install this project with:


```sh
  cd budget_app
  bundle install
  npm install
```


### Usage

To run the project, execute the following command:


```sh
  rails server
```


### Run tests

To run tests, run the following command:

```sh
  rspec spec
```


### Deployment

You can deploy this project using:


 <a href="https://render.com/">Visit Render</a>
  
  by following the docs instructions
